Clase 5 - Ejemplos de <code> - Texto en HTML

La etiqueta <code>está pensada para etiquetar fragmento de código de ordenador.
Los navegadores suelen mostrar la etiqueta <code> en tipo de letra monospace.

Ejemplo:

Con <code>git config --global core.editor “code --wait”</code> establecerás Visual Code como tu editor predeterminado en Git

Se vería así:

Con git config --global core.editor “code --wait” establecerás Visual Code como tu editor predeterminado en Git
