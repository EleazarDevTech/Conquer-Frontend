Ejemplo de etiquetas en HTML

  • Ejemplos de <hn>: Se usa para mostrar titulos
  • Ejemplos <p>: Envuelve los parrafos de texto
  • Ejemplo de enlaces <a>: Enlazara a otros archivos html o webs externas
  • Ejemplos <blockquote>: Lo usaremos para citar texto externo
  • Ejemplo <cite>: Se usa para citar una obra, un libro, un autor...
  • Ejemplo <code>: Se usa para mostrar código
  • Ejemplo <em>: Énfasis. Normalmente se muestra como cursiva
  • Ejemplo <strong>: Marca texto como importante
  • Ejemplo de entidades: Servirán para incluir carácteres especiales a una web
